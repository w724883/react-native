## 开发环境

`windows` `android`

## 配置环境

按照文档配置适合的环境[文档](http://reactnative.cn/docs/0.44/getting-started.html#content)

## 启动

启动 android studio

启动 android 模拟器

```
npm install

react-native run-android
```
